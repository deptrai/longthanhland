## 12. Cost Optimization Strategies

### Potential Savings

1. **Remove Project Manager**: Save 96,000,000 VND ($3,840)
   - Lead Developer handles PM duties

2. **Use Junior Developers**: Save ~30% on labor
   - Replace some Mid-Level with Junior for simple tasks
   - Estimated savings: 50,000,000 VND ($2,000)

3. **Reduce AI API Costs**: Save ~50% on OpenAI
   - Use GPT-3.5-turbo instead of GPT-4 where possible
   - Estimated savings: 20,000,000 VND ($800)

4. **Self-host Infrastructure**: Save ~40% on hosting
   - Use VPS instead of managed services
   - Estimated savings: 56,000,000 VND ($2,240)

5. **Phased Rollout**: Delay Phase 4
   - Launch MVP first, add marketplace later
   - Spread costs over longer timeline

**Total Potential Savings**: ~222,000,000 VND ($8,880)

**Minimum Viable Budget**: ~429,383,000 VND ($17,175)
